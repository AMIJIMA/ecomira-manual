# エコミラ LTE通信設定マニュアル
Version 1.0
2026-03
---

# 1. 概要

LTE通信モジュールにSIMカードを挿入し、  
モバイル通信を利用するための設定手順です。

SIMカードを交換した場合は  
**APN設定を含めすべての設定を再実施する必要があります。**

---

# 2. 対象機器

| 項目 | 内容 |
|-----|-----|
| LTEモジュール | Quectel LTE Module |
| 設定端末 | Windows PC |
| 接続方法 | USB接続 |
| AT通信 | Quectel USB AT Port |

---

# 3. 動作確認済みSIM

|-----|------|------|

# 4. 事前準備

以下の準備を行います。

### 必要ソフト

- Tera Term  
または  
- シリアル通信が可能なターミナルソフト

### 接続先ポート

```
Quectel USB AT Port
```

### シリアル通信設定

| 項目 | 設定 |
|-----|-----|
| ボーレート | 9600 |
| データビット | 8 |
| パリティ | なし |
| ストップビット | 1 |

# 5. 設定手順

---

# STEP 1  
WindowsでAPN設定を行う

1  
画面右下の  
**ネットワークアイコン**をクリック

2  
**携帯電話ネットワーク**を選択

3  
右側の設定画面から  

```
携帯電話
↓
詳細オプション
```

を開く

4  
**インターネットAPNの追加**

をクリック

5  
SIMに対応したAPNを入力

6  
保存をクリック

7  
追加したAPNが  

```
適用済み
```

となっていることを確認

---

### 接続確認

この時点で

```
アクティブ化済み
または
接続済み
```

になれば設定完了です。

その場合  
**STEP2以降は不要です。**

---

### 次のSTEPへ進む条件

APN状態が

```
適用済み
```

のまま変わらない場合は  
STEP2へ進みます。

---

# STEP 2  
ATコマンドでPDP APN設定を行う

TeraTermなどのターミナルソフトを起動し  

```
Quectel USB AT Port
```

へ接続します。

---

## ① 接続確認

次のコマンドを入力

```
AT
```

正常な場合

```
OK
```

が返ります。

---

### OKが返らない場合

以下を確認

- シリアルポート設定
- ボーレート9600

---

## ② 現在のAPN設定確認

```
AT+CGDCONT?
```

現在登録されているAPN一覧が表示されます。

**1番のAPN**がSIMのAPNと一致しているか確認します。

---

## ③ APN設定（必要な場合）

APNが一致していない場合のみ設定します。

```
AT+CGDCONT=1,"IPV4V6","APN"
```

APN部分にSIMのAPNを入力します。

---

### APN一覧

| SIM | APN |
|-----|-----|
| mineo | mineo.jp |
| MEEQ | meeq.io |

---

正常な場合

```
OK
```

が返ります。

ERRORが出る場合  
コマンド入力内容を確認してください。

---

## ④ LTEモデム再起動

次のコマンドを入力

```
AT+QPOWD=1
```

LTEモジュールが再起動します。

再起動には  
**約30秒〜60秒**かかります。

ターミナルが

```
切断されました
```

と表示されても  
そのまま待機してください。

---

# STEP 3  
接続確認

STEP1で開いていた

```
携帯電話設定画面
```

を確認します。

再度

```
携帯電話
```

を選択します。

---

### 接続状態

以下の表示になれば接続成功です。

```
接続済み
```

※アンテナ表示が0本でも通信可能な場合があります。

---

# 6. トラブルシューティング

| 症状 | 対処 |
|-----|-----|
| ATコマンドが反応しない | シリアルポート設定確認 |
| OKが返らない | ボーレート9600確認 |
| APN設定後も接続できない | APN再確認 |
| auSIMが他キャリアに接続 | COPSコマンドで解除 |

---

### キャリア接続リセット

```
AT+COPS=2
```

↓

```
AT+COPS=0
```

---

# 7. 注意事項

SIMカードを変更した場合

```
APN設定
ATコマンド設定
モデム再起動
```

を**すべて再実施**してください。
