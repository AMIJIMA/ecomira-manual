# エコミラ マニュアル

このリポジトリは、株式会社HRが開発した
空調省エネ制御装置 **エコミラ（ecomira）** の
社内技術マニュアルをまとめたものです。

エコミラの

* 機器設定
* 設置工事
* 通信設定
* 運用方法
* トラブル対応

など、現場で必要な情報を体系的に整理しています。

---

# マニュアル構成

```
エコミラ マニュアル

├ 基本
├ 設定
├ 工事
├ 運用
├ 通信
├ PC設定
├ トラブル対応
└ 開発資料
```
